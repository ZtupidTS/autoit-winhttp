Function Reference

_WinHttpGetDefaultProxyConfiguration

Retrieves the default WinHttp proxy configuration.

Syntax

#include "WinHttp.au3"
_WinHttpGetDefaultProxyConfiguration()

Parameters

 None.


Return Value

Success - Returns array with 3 elements:
                    $array[0] - Integer. Access type.
                    $array[1] - String. Proxy server list.
                    $array[2] - String. Proxy bypass list.
Failure - Returns 0 and sets @error:
                    1 - DllCall failed

Remarks

Access types are defined in WinHttpconstants.au3:
       $WINHTTP_ACCESS_TYPE_DEFAULT_PROXY = 0
       $WINHTTP_ACCESS_TYPE_NO_PROXY = 1
       $WINHTTP_ACCESS_TYPE_NAMED_PROXY = 3


Related

_WinHttpDetectAutoProxyConfigUrl, _WinHttpGetIEProxyConfigForCurrentUser, _WinHttpSetDefaultProxyConfiguration

See Also

MSDN

Example

#include "WinHttp.au3"
#include <Array.au3>

Opt("MustDeclareVars", 1)

; Current WinHTTP proxy configuration:
Global $aProxy = _WinHttpGetDefaultProxyConfiguration()
_ArrayDisplay($aProxy, "Current WinHTTP proxy configuration")
