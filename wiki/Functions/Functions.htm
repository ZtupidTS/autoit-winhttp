WinHttp Function Reference

Below is a complete list of the Functions available in WinHttp.
Click on a function name for a detailed description.

 

WinHttp Function Description
_WinHttpAddRequestHeaders Adds one or more HTTP request headers to the HTTP request handle.
_WinHttpBinaryConcat Concatenates two binary data returned by _WinHttpReadData() in binary mode.
_WinHttpCheckPlatform Determines whether the current platform is supported by this version of Microsoft Windows HTTP Services (WinHTTP).
_WinHttpCloseHandle Closes a single handle.
_WinHttpConnect Specifies the initial target server of an HTTP request and returns connection handle to an HTTP session for that initial target.
_WinHttpCrackUrl Separates a URL into its component parts such as host name and path.
_WinHttpCreateUrl Creates a URL from array of components such as the host name and path.
_WinHttpDetectAutoProxyConfigUrl Finds the URL for the Proxy Auto-Configuration (PAC) file.
_WinHttpGetDefaultProxyConfiguration Retrieves the default WinHttp proxy configuration.
_WinHttpGetIEProxyConfigForCurrentUser Retrieves the Internet Explorer proxy configuration for the current user.
_WinHttpOpen Initializes the use of WinHttp functions and returns a WinHttp-session handle.
_WinHttpOpenRequest Creates an HTTP request handle.
_WinHttpQueryDataAvailable Returns the availability to be read with _WinHttpReadData().
_WinHttpQueryHeaders Retrieves header information associated with an HTTP request.
_WinHttpQueryOption Queries an Internet option on the specified handle.
_WinHttpReadData Reads data from a handle opened by the _WinHttpOpenRequest() function.
_WinHttpReceiveResponse Waits to receive the response to an HTTP request initiated by WinHttpSendRequest().
_WinHttpSendRequest Sends the specified request to the HTTP server.
_WinHttpSetCredentials Passes the required authorization credentials to the server.
_WinHttpSetDefaultProxyConfiguration Sets the default WinHttp proxy configuration.
_WinHttpSetOption Sets an Internet option.
_WinHttpSetStatusCallback Sets up a callback function that WinHttp can call as progress is made during an operation.
_WinHttpSetTimeouts Sets time-outs involved with HTTP transactions.
_WinHttpSimpleFormFill Fills form.
_WinHttpSimpleReadData Reads data from a request
_WinHttpSimpleReadDataAsync Reads data from a request in asynchronous mode
_WinHttpSimpleRequest A function to send a request in a simpler form
_WinHttpSimpleSendRequest A function to send a request in a simpler form, but not read the data
_WinHttpSimpleSendSSLRequest A function to send a SSL request in a simpler form, but not read the data
_WinHttpSimpleSSLRequest A function to send a SSL request in a simpler form
_WinHttpTimeFromSystemTime Formats a system date and time according to the HTTP version 1.0 specification.
_WinHttpTimeToSystemTime Takes an HTTP time/date string and converts it to array (SYSTEMTIME structure values).
_WinHttpWriteData Writes request data to an HTTP server.
