Function Reference

_WinHttpTimeToSystemTime

Takes an HTTP time/date string and converts it to array (SYSTEMTIME structure values).

Syntax

#include "WinHttp.au3"
_WinHttpTimeToSystemTime($sHttpTime)

Parameters

$sHttpTime Date/time string to convert.

Return Value

Success - Returns array with 8 elements:
                    $array[0] - is Year,
                    $array[1] - is Month,
                    $array[2] - is DayOfWeek,
                    $array[3] - is Day,
                    $array[4] - is Hour,
                    $array[5] - is Minute,
                    $array[6] - is Second.,
                    $array[7] - is Milliseconds.
Failure - Returns 0 and sets @error:
                    1 - DllCall failed

Related

_WinHttpTimeFromSystemTime

Example

#include "WinHttp.au3"
#include <Array.au3>

Opt("MustDeclareVars", 1)

; Time
Global $aTime = _WinHttpTimeToSystemTime("Sat, 21 Aug 2010 22:04:43 GMT")
_ArrayDisplay($aTime, "_WinHttpTimeToSystemTime()")
